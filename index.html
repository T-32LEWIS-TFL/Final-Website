EIGHT TYPOGRAPHIC SYSTEMS

AXIAL This is my example of an Axial typographic system. An Axial typographic system refers to a design approach where text elements are arranged along a central axis or axes. This system is characterised by a strong emphasis on symmetry and balance. The central axis serves as a guide for arranging and aligning text elements, creating a structured and organised layout.

RADIAL This is my example of a Radial typographic system. A Radial typographic system is a design approach where text elements are arranged in a radiating pattern, emanating from a central point. This system is characterised by a dynamic and often unconventional layout that draws attention to the centre and creates a sense of movement.

DILATATIONAL This is my example of a Dilatational typographic system. A Dilatational typographic system is a design approach where text elements are arranged in a circular pattern circling a single point. Additional text usual being stacked atop each other, still in the circular shape. This system is characterised by there being multiple circles to allow different texts to be separated and aligned at different place.

RANDOM This is my example of a Random typographic system. A Random typographic system refers to a design approach where text elements are placed without a strict or predetermined order, randomly. Often, the text of this typographic system is dispersed and usually placed as if a random generator was deciding the text placement, text often very small, and very large, that also overlay each other.

GRID This is my example of a Grid typographic system. A Grid typographic system involves the organising and aligning of text elements on a structured grid. This system relies on a set of horizontal and vertical lines that create a framework for the placement of text. Grid-based typography focuses on ensuring consistency and readability.

MODULAR This is my example of a Modular typographic system. A Modular typographic system is one that has paragraphs and text ordered and organised into blocks or modules that usually have some sort of container/box shape, easily separating the text from the background and each module/block from each other. The Modular typographic system focuses on readability and consistency but is structured in a way that things are not necessarily read from top to bottom, left to right.

TRANSITIONAL This is my example of Transitional typographic system. A Transitional typographic system involves the placement and appearance of the text appearing to be that of movement or of something that is moving. The text of the Transitional typographic system tends to prioritise what the shape of the inferred move object is just as much as what the text itself reads and is one of the more creative of the typographic systems.
